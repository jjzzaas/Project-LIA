# 챕터 수정 안내

이 폴더는 비주얼 노벨의 대사, 선택지, 호감도 결과를 관리합니다.

## 파일 찾는 방법

- 챕터 1: `001-010/chapter-001.js`
- 챕터 11: `011-020/chapter-011.js`
- 챕터 100: `091-100/chapter-100.js`

## 직접 수정해도 되는 부분

```js
text: '화면에 나오는 대사'
label: '플레이어가 누르는 선택지'
affection: { haru: 2 }
```

대사나 선택지 문구는 따옴표 안쪽만 바꾸면 됩니다.

## 선택지 작성 원칙

모든 선택지는 아래 중 하나 이상의 결과를 가져야 합니다.

- 캐릭터 호감도 변화
- 신뢰도 또는 성향 변화
- 바로 다음 반응 변화
- 이후 대사 변화
- 이벤트·분기·엔딩 조건 기록

결과가 완전히 같은 선택지는 만들지 않습니다.

## 건드리지 않는 것이 좋은 부분

- `{ } [ ] , :` 같은 기호
- `id` 값
- `export` 문장
- 엔진과 연결되는 속성 이름

문장 하나를 고칠 때는 GitHub 검색에서 기존 대사의 일부를 검색하면 해당 장면을 바로 찾을 수 있습니다.
